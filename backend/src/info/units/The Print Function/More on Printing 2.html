The print function also accepts named arguments, such as `sep` and `end`

For example, the following code prints the numbers 1 through 5, separated by hyphens:

print(1, 2, 3, 4, 5, sep="-") # prints: 1-2-3-4-5

The `end` parameter controls what is printed at the end of the print statement. Print statements by default end with a newline character, but you can change that by passing the `end` parameter:

print("Hello", end=" World") # prints: Hello World

both arguments are optional, and can be used together:

print(1, 2, 3, 4, 5, sep="-", end="!") # prints: 1-2-3-4-5!

Try printing the lower-case letters 'a', 'b', 'c' and 'd'; seperated by `->`, and ending with `!!`. The output should look like:

a->b->c->d!!
