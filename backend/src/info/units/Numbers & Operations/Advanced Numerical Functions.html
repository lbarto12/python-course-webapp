There are two more complex functions that can be performed on numbers:

the `divmod` function divides the first argument by the second argument and returns the `quotient` and the `remainder` as a tuple.

x = divmod(5, 2) print(x) # prints: (2, 1)

And finally the `bin` function returns the binary representation of a number. While not often used, it will be useful in future modules.

x = bin(20) print(x) # prints: 0b10100

Given `a` and `b`, `unpack` the result of `divmod` on `a` and `b` into two variables and print them on the same line. Then print the binary representation of `b`.
