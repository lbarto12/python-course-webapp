There are some more complex functions that can be performed on numbers:

The `abs` function, this function is used to find the absolute value of a number.

print(abs(-1)) # prints: 1

The `round` function, this function is used to round a number to the nearest integer.

print(round(1.5)) # prints: 2

This function can also be used to round a number to a certain number of decimal places.

print(round(1.2345, 2)) # prints: 1.23
