Numbers can be modified and assigned to the result of an operation in one line. This is achieved by using the `=` operator immediately after an operator.

For example, the following code will add `5` to `a` and assign the result to `a`:

a = 5 a += 5 print(a) # prints: 10

This works for all the operators we have discussed so far. even the complex ones.

a = 5 a **= 5 print(a) # prints: 3125 a //= 5 print(a) # prints: 625 a %= 5 print(a) # prints: 0

Given `a` and `b`, perform the following operations on `a` and assign the result to `a`:

a %= b a **= b a //= b
