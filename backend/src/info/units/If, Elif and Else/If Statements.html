The `if` statement is the fundemental building block of all programming. It is what allows us to change a program's behaviour based on the values of variables.

`if` statements are predicated upon boolean values. When the boolean value is `True`, the code inside the `if` statement is executed. When the boolean value is `False`, the code inside the `if` statement is skipped.

is_true = True if is_true == True: print("This code is executed")

You can also do comparisons `inline`:

a, b = 5, 5 if a == b: print("This code is executed")

Given `a` and `b`, try printing `A is greater` if `a` is greater than `b`. Otherwise, print nothing.
