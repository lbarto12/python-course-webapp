There are various `Boolean Operators` that can be used to check multiple things at once. These are the basic ones:

and - Returns `True` if both statements are `True` or - Returns `True` if one or both of the statements is `True` not - Reverse the result, returns `False` if the result is `True` and vice versa

In practice, there is a more concise way of phrasing this, but here is an example of how to use `and`:

a, b = True, False if a and b: print(True) else: print(False)

Given two boolean values `a` and `b`, print `True` if both are `True`, and `False` otherwise.
