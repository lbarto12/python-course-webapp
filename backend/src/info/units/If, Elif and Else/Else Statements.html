The `else` statement is used to execute code when the `if` statement is `False`. Else statements are optional, and don't require a condition.

if 5 > 2: print("Five is greater than two!") else: print("Five is not greater than two!")

An `important note` is that else will execute in ANY case where the `if` statement is `False`. in the following code, the else statement will print `B is greater` even though the numbers compared are the same value.

a, b = 5, 5 if a > b: print("A is greater") else: print("B is greater")
