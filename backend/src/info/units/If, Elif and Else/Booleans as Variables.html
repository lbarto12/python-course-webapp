We can also store `Booleans` in variables. This works much the same as creating a variable for a number or string.

is_true = True

We can also `initialize` boolean variables by comparing two values. For example, we can compare two numbers to see if one is greater than the other:

is_greater = 10 > 5 # is_greater is True

Given two numbers `a` and `b`, try storing whether `a` is `less than` than `b` in a variable, then print the variable.
