`Type Casting` refers to changing an entity of one data type into another. Basic type casting can be very useful, and is sometimes required when working with different data types or user input.

for example, user input is always a string, but sometimes we need to use it as a number. to do so, we can use the `int()` function.

response = input("what is your age?") response = int(response)

Most types can be cast into other types, irrespective of where they come from or are stored

x = "20.5" y = float(x) z = int(x)

Some casts have different behaviour depending on the type of the entity being cast. take, for example, casting a `boolean` to an `int`. if the boolean is `True`, the result will be `1`, and if it is `False`, the result will be `0`.

x = True y = int(x) print(y) # prints 1

This occurs because `True` and `False` are actually just `1` and `0` in disguise. we can even use them as numbers to prove this point

x = True y = True z = x + y print(z) # prints 2
