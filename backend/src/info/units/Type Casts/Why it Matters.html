`Type Casting` is mostly important when combining and comparing different data types.

For example, if you try to add a string and a number, you will get an error.

x = "20" y = 5 z = x + y # error

Even more confusingly, if you try to `multiply` a `string` by a `number`, you will actually get a valid but completely different interpretation of the expression:

x = "20" y = 5 print(x * y) # prints: 2020202020 print(int(x) * y) # prints: 100

What is most important to remember, is that if you are unsure of the `type` of a variable, you probably haven't `typed` your variables well enough. Remember too, if you need to be sure, you can always use the `type()` function to find out.
