lists are a collection of items, each item has a unique index starting from 0 to the `length` of the list - 1.

lists are defined using square brackets `[ ]` and each item is separated by a comma `,`:

my_list = [1, 2, 3, 4, 5]

you can access an item in the list by referencing its index inside square brackets `[ ]`:

print(my_list[0]) # prints: 1 print(my_list[2]) # prints: 3

Lists are also `mutable`, meaning that you can change the value of an item in the list by referencing its index:

my_list[0] = 10 print(my_list) # prints: [10, 2, 3, 4, 5]

Typing for lists is most adequately defined in the typing module as `List`, which takes a `type` argument for the type of the items in the list:

from typing import List my_list: List[int] = [1, 2, 3, 4, 5]

Given the list `my_list`, try changing the value of the item at `index 2` to the value `10`, then `print the list`.
