`Dictionaries` are arguably one of the most useful data types in Python. They are a collection of `key-value` pairs, where each `key` is `unique` and each `value` is associated with a `key`.

To declare a `dictionary`, we use curly braces `{ }` and separate the `key-value` pairs with commas `,`:

x = {'a': 1, 'b': 2, 'c': 3}

To access a value in a `dictionary`, we use the `key` associated with that value:

x = {'a': 1, 'b': 2, 'c': 3} print(x['a']) # prints: 1

To add a new `key-value` pair to, or change a value in a `dictionary`, we use the `key` and assign it a `value`:

x = {'a': 1, 'b': 2, 'c': 3} x['d'] = 4 print(x) # prints: {'a': 1, 'b': 2, 'c': 3, 'd': 4}
