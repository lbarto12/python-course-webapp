`Sets` are collections of `distinct` values. They are mutable, which means that once declared, they can be changed.

To declare a `set`, we use curly braces `{ }` and separate the values with commas `,`:

x = {1, 2, 3}

However, if you try to declare a set with no elements, you will get an `Error` when trying to perform operations on it. This is because when no elements are present, Python will interpret it as a `Dictionary` instead. We will talk about dictionaries on the next slide.

To declare an empty set, we use the `set()` function:

x = set()

The most important distintion `sets` and other `iterative types` is that `sets` are `unordered`, and only hold `distinct values`. This means that the order of the elements in the set is not guaranteed, and that no two elements in the set can be the same.

`sets` with duplicate values will be reduced to a single value:

x = {1, 2, 2, 3, 3} print(x) # prints: {1, 2, 3}
