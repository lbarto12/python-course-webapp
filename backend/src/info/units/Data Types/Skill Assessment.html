These are the `data types` we have covered so far:

Numbers, Strings, Booleans, Lists, Tuples, Sets, Dictionaries

Now we will test your knowledge of these `data types` with a `skill assessment`. for each answer, print the answer on a new line.

1. Create a two `numbers` one `int` and one `float` such that the answer sums to 5.25

2. Create two `strings` that contains the words `Hello` and `World` respectively and print them such that the answer is `Hello World` on the same line.

3. Create a `boolean` that is `True` and print it.

4. Create a `list` that contains the numbers `1, 2, 3, 4, 5`, change the 3rd element to 20 and print the `list`.

5. Create a `tuple` that contains the numbers `1, 2, 3`, `unpack` it and print the values on the same line.

6. Create a `set` that contains 7 numbers, but only ends up with the numbers `1, 2 and 3` inside it, print the `set`.

7. Create a `dictionary` that contains the `key-value` pairs `{'a': 1, 'b': 2, 'c': 3}`, add the `key-value` pair `{'d': 4}` to the `dictionary` and print the `dictionary`.
